Abstract

The rapid development of web agents has led to increasing reliance from both individuals and enterprises. While large language and vision-language models that power these agents are predominantly text-driven, most robustness studies have focused narrowly on text-based vulnerabilities such as prompt injection. However, as visual input becomes an integral component of web agents, the influence of visual elements on their behavior remains underexplored. In this work, we introduce WebAgentsLook, a systematic pipeline that automatically generates HTML variants by modifying CSS attributes, including color, typography, position, and their combinations. By evaluating multiple web agents on these variants, we uncover novel insights into how visual elements guide or mislead agent decisions. Our evaluation combines coordinate-based action tracing with semantic analysis, and we propose a comprehensive metric to quantify task success, defined by whether the agent correctly selects the intended target.

Method Overview

Open PDF in new tab

Our pipeline consists of three main phases:

  1. Variant Generation: Automatically generate HTML variants by modifying CSS attributes (color, position, typography, size) while preserving semantic content.
  2. Realistic Browsing Simulation: Simulate realistic web browsing with viewport-based scrolling and interaction, mirroring how humans navigate web pages.
  3. Dual Evaluation: Assess agent behavior using both coordinate-based click accuracy and semantic understanding metrics.

Key Findings

🎨

Color Influence

Color variations exert the strongest influence on agent choices. Agents show significant sensitivity to color changes, similar to human visual attention patterns.

📍

Positional Bias

Agents exhibit strong positional bias, preferring early-appearing items on a page. Moving target items to lower positions dramatically reduces selection probability.

🔍

Text-Dominant Attention

Current VLM-based agents tend to focus disproportionately on textual/OCR signals, showing weaker robustness to purely visual changes compared to humans.
