## Image Optimization Tips

1. **Format**: Use PNG for diagrams with text, JPEG for photographs
2. **Compression**: Optimize with tools like TinyPNG or ImageOptim
3. **Resolution**: Use 2x resolution for retina displays
4. **File Size**: Keep under 500KB per image for fast loading
